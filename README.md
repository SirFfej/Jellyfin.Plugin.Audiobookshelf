This project is in Extreme Alpha Test mode.

***Use at your own risk.***

Intended to integrate the metadata and chapter data from Audiobookshelf service with the Jellyfin service so that the Jellyfin user interface can be a unified front end.

Jellyfin is not specially suited for media beyond video types. This is likeluy the first of a series of addons intended to enhance the jellyfin media use experience so that is can remain a unified user experience.
